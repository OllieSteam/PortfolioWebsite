My Missile Flight Path Simulator Project

This code is a simulation of a missile's flight path from its initial position, velocity, and acceleration to its target position using the Euler method. The simulation is implemented using Python programming language and utilizes several libraries such as matplotlib, mpl_toolkits, and numpy.

The matplotlib library is used to create the 3D plot that displays the target position as a red dot and the missile's flight path as blue dots. The mpl_toolkits library is used to enable the use of Axes3D for the plot.

The simulation is initialized with the target position, initial position, velocity, and acceleration of the missile. The fuel burn rate, maximum range, and time step of the simulation are also defined. A while loop is used to simulate the flight of the missile, updating the position, velocity, and acceleration at each iteration. The fuel level is also decreased at each iteration. The missile's position is stored in a list at each time step.

The simulation ends when the fuel level reaches zero or the maximum range is reached. The code uses display() function to display the 3D plot, with the target as plot.

Overall, this code is a implementation of a missile flight path simulation using Python programming language and various libraries. It utilizes the Euler method for updating the missile's position, velocity and acceleration, and uses matplotlib to create an informative 3D plot to visualize the simulation.

This is loaded dynamically with a pyscript library.

Stock Market Simulator

The script starts by importing the matplotlib library and the random module. Next, I defined an empty array called stock_array that will be used to store the stock values generated by the simulator.

The script then defines several functions:

  • generate_graph() function takes in an array of stock values and a stock name as input, then generates a graph of the stock data using matplotlib. The function sets the x-axis as the number of runs and the y-axis as the price of the stock. It also annotates the final value of the stock on the graph and adds a horizontal line at the final value of the stock.

  • generate_stock_value() function takes in upper and lower bounds, a volatility base value, and the current stock value as input, and generates a new stock value. The function uses the upper and lower bounds to calculate a percentile value, which is then used to adjust the volatility base value.

  • generate_stock() function takes in upper and lower bounds, a volatility base value, an initial stock value, the number of runs, and the stock array as input. It uses a for loop to run the generate_stock_value() function the specified number of times and append the new stock value to the stock_array.

  • average() function takes in an array as input and calculates the average value of the array.

  • final_constructor function takes the stock name, upper bounds, lower bounds, volatility base values and stock values and returns an array of the final data to be used by the simulator.

Finally, the script reads in the 'stock_data.data' file and processes the data to create arrays of stock names, upper bounds, lower bounds, volatility base values, and initial stock values. These arrays are then passed to the final_constructor function to create the final data set used by the simulator. The script uses these functions to process the data, calculate new stock values, generate the final data set and also generates a graph of the stock data using matplotlib.
